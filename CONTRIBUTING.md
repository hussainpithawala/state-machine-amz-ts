# Contributing to state-machine-amz-ts

## Setup

```bash
git clone https://github.com/hussainpithawala/state-machine-amz-ts.git
cd state-machine-amz-ts
npm install
```

## Development

```bash
make test          # run tests
make lint          # lint source
make typecheck     # type check
make build         # build dist
make all           # full pipeline
```
