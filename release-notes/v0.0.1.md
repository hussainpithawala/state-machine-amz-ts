# v0.0.1 — Initial Release

## What's included
- `StateMachine` — in-memory execution engine (Amazon States Language compatible)
- `PersistentStateMachine` — execution engine with pluggable repository
- `PostgresRepository` — stub, to be implemented
- `Executor` — task handler registry
- All ASL state types scaffolded (Task, Pass, Choice, Wait, Succeed, Fail, Parallel, Map)
- Full TypeScript types for executions, state history, filters

## What's coming
- Full ASL execution engine (all state types)
- PostgreSQL repository implementation
- Redis-backed distributed queue
- Message correlation (pause/resume on external events)
- Execution chaining
- Micro-batch orchestration
